# Quickstart: deep-ai-assist

## Prerequisites

1.  A functional macOS desktop environment.
2.  A connected webcam.
3.  The `GEMINI_API_KEY` environment variable must be set.
4.  Audio output enabled (for voice prompts).

## Setup

Ensure dependencies are installed:

```bash
uv pip install google-genai
# OpenCV and Ultralytics should already be installed from previous cycles
```

## Validation Scenario 1: Proactive Voice Alert

1.  Start the application with deep AI assist enabled:
    ```bash
    uv run python -m src.ui.camera_app --deep-ai-enabled
    ```
2.  Point the webcam at a distinct scene (e.g., heavily backlit subject, or a very dark room).
3.  Wait for approximately 10 seconds (the polling interval for the cloud AI).
4.  **Expected Outcome 1**: You should hear a voice prompt generated by the macOS `say` command (e.g., "It looks quite dark here, I'm increasing the ISO for you.").
5.  **Expected Outcome 2**: You should see a popup on the UI overlay displaying the `SceneContext` (e.g., "Scene: Low Light, ISO: 1600").
6.  **Expected Outcome 3**: The camera feed should visually brighten (either via hardware exposure adjustment or software exposure simulation) without significantly dropping the frame rate of the live feed below 25 FPS.

## Validation Scenario 2: Interactive Query Mechanism

1.  While the application is running, press the `i` key on your keyboard to enter Input Mode.
2.  An input box should appear at the bottom of the screen: "向AI提问: _"
3.  Type a question, for example: "光线太暗怎么办" (What should I do if it's too dark?)
4.  Press the `Enter` key to submit the query.
5.  **Expected Outcome**: The AI Coach should process your specific question along with the current frame, and return a tailored response displayed as a Gold popup (chat response).
