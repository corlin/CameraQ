# Data Model: Generative AI Guide (Stage 4)

## Entities

### `AICoachingResult`

Represents the parsed response from the AI coaching service.

| Field | Type | Description |
|-------|------|-------------|
| `advice_text` | `str` | The high-level stylistic or emotional photography advice generated by the LLM. |
| `timestamp` | `float` | The `time.time()` when the result was generated. Used for UI rendering expiration. |
| `is_error` | `bool` | True if the network request failed or timed out. |

## Integration with `AnalysisResult`

The existing `AnalysisResult` in `src/core/entities.py` will be extended to include:

| Field | Type | Description |
|-------|------|-------------|
| `ai_coaching` | `Optional[AICoachingResult]` | The most recent AI coaching advice to be rendered by the UI. |
